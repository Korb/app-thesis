Yokatta© — это приложение с открытым исходным кодом для изучения языка с использованием карточек и интервального повторения.

Алгоритм во многом основан на SM-2 SuperMemo и на момент последнего релиза соответствовал лучшим практикам использования компонентов архитектуры Android, предоставленных Google.

<b>Возможности приложения</b>
* выбор цветовых тем;
* случайный порядок карточек;
* анимация повторения для визуальной обратной связи;
* эффект искажения уведомления при не совсем верном ответе;
* тактильная обратная связь при отправке пустого ответа;
* аудиозапись для последующей самостоятельной оценки произношения;
* двойное нажатие «Назад» для выхода из режима повторения;
* автофокусировка поля ввода;
* отображение диаграмм прогресса;
* смахивание для изменения или удаления карточек;
* поиск по карточке;
* экспорт карточек в JSON.

Исходный код приложения — https://github.com/uestueab/app-thesis
